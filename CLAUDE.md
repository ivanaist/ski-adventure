# Ski Adventure

A browser skiing game built by Eero (age 10) for his little sister Emma (age 6), 
with help from aunt Ivana using Claude Code.

## Tech Stack
- Single file: index.html (HTML + CSS + JavaScript, no frameworks)
- No external dependencies

## Rules
- ALWAYS keep everything in one index.html file
- NEVER use frameworks like React or Vue
- Keep code simple and readable — this is a learning project
- After every significant feature, remind Ivana to save to GitHub
- Keep gameplay simple — this is designed for a 6-year-old
- Controls must be minimal: one or two keys maximum
- Visuals must be colorful, friendly and bold — no tiny elements

## Deployment
- Deploy via GitHub Pages
- Repository: https://github.com/ivanaist/ski-adventure

## How to run
Open index.html in any browser.
